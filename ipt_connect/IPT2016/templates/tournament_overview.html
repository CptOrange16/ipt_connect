Welcome !

This site regroups all the informations relative to the IPT2016 scores: grading, points distribution, statistics and many more ! This is currently a beta version, not fully functional, and likely with a few bugs here and there. If you see anything suspicious, if you want to comment the relevance of some elements, or if you would like to see something added to this site, please email me at president@iptnet.info.

The current page "Home" regroups the main informations about the tournament. Use the top-menu to navigate through the various sections. A lot of small elements are clickable (e.g. team names, problem names, some grades, the physics fights, etc...), do not miss them ! The page "Physics Fight" displays a summary of all the physics fights. The page "Ranking" ranks the teams from best to worst. The page "Problem" displays all the problems of this edition, and the page "Participants" displays the participants ranked by the number of points they gathered (not multiplied by the multiplicative coefficients). Each of this page has (of will have in a near future) some sub-pages that you can discover using your mouse by clicking everywhere.

Known bugs : The ranking, participants and individual team pages are slow to load (up to 3-4 sec. on my machine), and repeateadly loading the page may crash the server. Clicking of participant and jury members name return a nearly empty page.
